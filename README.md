## Getting Started

Installation is seamless and easy using `pip`.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:AnnaVo0/HealthBit.git
   ```
2. Install packages
   ```sh
   pip install -r requirements.txt
   ```
3. Run a local dev server to check out the project
   ```
   python app.py
   ```
   or
   ```
   flask run
   ```
